# SmartSpend: AI-Powered Personal Finance Assistant

SmartSpend is a personal finance and budgeting application designed to help users better understand how they spend their money. 
By uploading bank statements, users can automatically categorize transactions, generate monthly spending summaries, and receive AI-generated insights and recommendations tailored to their habits.

## Features
- Upload bank statements (CSV format)
- Automatic categorization of transactions using AI
- Monthly spending reports with visual breakdowns
- Spending alerts when budgets are exceeded
- Search and filter transactions by date, category, or keywords
- Secure login and account management
- AI-generated insights and savings recommendations

## AI Integration
SmartSpend uses Google Gemini API for:
- Classifying transaction descriptions into categories
- Generating personalized insights and saving strategies

API Used: https://aistudio.google.com/welcome
